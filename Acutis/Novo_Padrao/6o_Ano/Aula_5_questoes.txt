O que são substantivos biformes?
A. Nomes que possuem uma única forma invariável para ambos os sexos
B. Nomes que possuem duas formas distintas, uma para o masculino e outra para o feminino
C. Nomes que mudam de significado quando se troca a vogal temática
D. Nomes usados exclusivamente para animais selvagens
ANSWER: B

Como se classificam os substantivos uniformes como "a cobra", que possuem apenas uma forma gramatical para nomear animais e exigem o acréscimo das palavras "macho" ou "fêmea"?
A. Sobrecomuns
B. Epicenos
C. Comuns de dois gêneros
D. Biformes heterônimos
ANSWER: B

A palavra "a criança" é um substantivo uniforme que se refere a pessoas e mantém a mesma forma e o mesmo artigo definido feminino ("a"), tanto para meninos quanto para meninas. Esse substantivo é classificado como:
A. Comum de dois gêneros
B. Epiceno
C. Sobrecomum
D. Biforme
ANSWER: C

Qual é a característica principal dos substantivos classificados como "comuns de dois gêneros", como em "o estudante / a estudante"?
A. A palavra muda de radical no feminino
B. A palavra é invariável, mas o gênero é indicado pela mudança do artigo ou determinante
C. Exigem o acréscimo dos termos "macho" ou "fêmea"
D. Não aceitam flexão de número no plural
ANSWER: B

Qual das opções a seguir apresenta um exemplo de substantivo biforme heterônimo (que usa radicais completamente diferentes para o masculino e o feminino)?
A. gato / gata
B. menino / menina
C. homem / mulher
D. duque / duquesa
ANSWER: C
